# 🏔️ Smart Trekking & Travel Packing Assistant

> Stop forgetting essentials. Start trekking smarter.  
> A destination-aware packing checklist app that generates your perfect gear list in seconds.

---

## ✨ Key Features

- **Smart List Generation** — Select your destination type (Snow/Trek, Hill/Hike, Terai/Safari) and number of days — the app generates a tailored packing checklist instantly.
- **Authentic Trekking Items** — Real gear included: Windcheater, Thermal wear, Trekking poles, Crampons, ORS packets, Trek permits, and more.
- **Animated Progress Bar** — Smoothly fills from 0% to 100% as you check off items. Shows exact count of packed vs remaining items.
- **Completion Message** — "🎒 All packed! Have a great trek!" appears when everything is checked.
- **Smooth Item Animations** — Each checklist item has a satisfying scale animation on check/uncheck.
- **LocalStorage Integration** — Your destination, days, and checked items are saved. Come back after a refresh — your list is right where you left it.
- **Reset Session** — One click wipes everything clean for your next trip.
- **Mobile-First Design** — Glassmorphism UI with nature-inspired colors. Looks great on any phone.

## 🚀 How to Run Locally

1. Clone the repository:
```bash
   git clone https://github.com/babluverse/trekking_assistant.git
```

2. Open the project folder:
```bash
   cd trekking_assistant
```

3. Open `index.html` directly in your browser — no setup, no dependencies needed.
